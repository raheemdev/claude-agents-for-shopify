# SOP: SEO

## On-Page Checklist

For every page (product, collection, blog post, custom page):

- [ ] **Title tag:** Under 60 characters, includes primary keyword, unique across site
- [ ] **Meta description:** 120-155 characters, includes primary keyword, has a call to action
- [ ] **H1:** One per page, includes primary keyword, different from title tag
- [ ] **Schema markup:** Product pages have Product schema, collections have CollectionPage, articles have Article
- [ ] **Images:** Descriptive filenames (not `IMG_4032.jpg`), alt text with keyword where natural
- [ ] **Internal links:** At least 2-3 relevant internal links per page
- [ ] **URL slug:** Short, keyword-rich, no stop words (e.g., `/collections/mens-jackets` not `/collections/the-best-mens-jackets-for-you`)

## Audit Cadence

| Task | Frequency | Tool |
|------|-----------|------|
| Full SEO audit (Lighthouse) | Monthly | `tools/scripts/seo-audit.sh` |
| Keyword ranking check | Bi-weekly | Ahrefs / Search Console |
| Broken link scan | Monthly | Screaming Frog or ahrefs |
| Index coverage review | Monthly | Google Search Console |
| Competitor keyword gap analysis | Quarterly | Ahrefs |

## Keyword Tracking Process

1. Maintain a keyword list in the client's `seo/keywords.csv` with columns: keyword, page, current rank, target rank, search volume.
2. Update rankings bi-weekly from Search Console or Ahrefs.
3. Flag any keyword that drops more than 5 positions for investigation.
4. Review keyword targets quarterly and adjust based on performance.

## Technical SEO Checks

Run these checks monthly alongside the Lighthouse audit:

- [ ] **Robots.txt:** Not blocking important pages or resources
- [ ] **Sitemap.xml:** Auto-generated by Shopify, verify it includes all important pages
- [ ] **Canonical tags:** Present on all pages, no self-referencing issues on paginated collections
- [ ] **Redirect chains:** No chains longer than 1 hop; fix double redirects
- [ ] **404 errors:** Review Search Console for crawl errors, create redirects for high-traffic 404s
- [ ] **Page speed:** Core Web Vitals passing (see `docs/SOPs/speed.md`)
- [ ] **Mobile usability:** No mobile usability errors in Search Console
- [ ] **Structured data:** Validate with Google Rich Results Test, fix any errors
- [ ] **Hreflang:** If multi-market, verify hreflang tags are correct
- [ ] **Duplicate content:** Check for duplicate title tags and meta descriptions
